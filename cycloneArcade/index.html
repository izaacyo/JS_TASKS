Cyclone Arcade Game

Instructions

  1. Create a number of bulbs and store them in an array
  2. Turn them all off by default
  3. Select the first one to turn on
  4. Switch off/turn on bulb after time Interval
  5. When end of the array, start again
  6. Add a button to stop the interval upon click
  7. Style the bulbs in a circle with JS Snippet
  8. Bonus points:
    1. Adjustable time interval
    2. Adjustable number of Bulbs
